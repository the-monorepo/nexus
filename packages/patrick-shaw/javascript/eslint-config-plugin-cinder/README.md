# Eslint Config Plugin Cinder

A pratical Cinder style preset for ESLint

## Installation

```bash
npm install --save-dev @pshaw/eslint-config-plugin-cinder
```
or
```bash
yarn add --dev @pshaw/eslint-config-plugin-cinder
```

# Tslint Preset-Core

A pratical style preset for TSLint

## Installation

```bash
npm install --save-dev @pshaw/tslint-preset-core tslint
```
or
```bash
yarn add --dev @pshaw/tslint-preset-core tslint
```

## Examples

```json
{
  "extends": [
    "@pshaw/tslint-preset-core"
  ]
}
```

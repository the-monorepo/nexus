# Eslint Config Plugin Typescript

A pratical Typescript style preset for ESLint

## Installation

```bash
npm install --save-dev @pshaw/eslint-config-plugin-typescript
```
or
```bash
yarn add --dev @pshaw/eslint-config-plugin-typescript
```

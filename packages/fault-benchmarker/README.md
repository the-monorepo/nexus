# Benchmarker

Benchmarking library for measuring various fault localization/repair algorithms in Fault.js

## Installation

```bash
npm install --save @fault/benchmarker@1.0.0-alpha.0
```
or
```bash
yarn add @fault/benchmarker@1.0.0-alpha.0
```

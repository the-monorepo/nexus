# Fl Istanbul Reporter

Use Mocha tests in the fl test runner

## Installation

```bash
npm install --save fl-istanbul-reporter@1.0.0-alpha.0
```
or
```bash
yarn add fl-istanbul-reporter@1.0.0-alpha.0
```

# Keyed Binary Heap

A implementation of a binary heap with O(1) search & item updates

## Installation

```bash
npm install --save @pshaw/keyed-binary-heap@1.0.0-alpha.0
```
or
```bash
yarn add @pshaw/keyed-binary-heap@1.0.0-alpha.0
```

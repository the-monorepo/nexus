# Writeme Core

A readme generator

## Installation

```bash
npm install --save-dev @writeme/core@3.1.4-0
```
or
```bash
yarn add --dev @writeme/core@3.1.4-0
```

## Examples

```javascript
import writeme from '@writeme/core';
// Point to where the directory where the package.json and writeme.config.js files are
writeme(__dirname);
```

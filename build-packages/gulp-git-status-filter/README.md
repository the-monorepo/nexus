# Gulp Git Status Filter

A gulp plugin that filters out files based off their git status

## Installation

```bash
npm install --save gulp-git-status-filter@1.0.0-alpha.1
```
or
```bash
yarn add gulp-git-status-filter@1.0.0-alpha.1
```

# Gulp Status Git Filter

A gulp plugin that filters out files based off their git status

## Installation

```bash
npm install --save gulp-status-git-filter@1.0.0-alpha.3
```
or
```bash
yarn add gulp-status-git-filter@1.0.0-alpha.3
```

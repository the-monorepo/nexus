# Gulp Status Git Filter

A gulp plugin that filters out files based off their git status

## Installation

```bash
npm install --save-dev gulp-status-git-filter@1.0.0-alpha.5
```
or
```bash
yarn add --dev gulp-status-git-filter@1.0.0-alpha.5
```

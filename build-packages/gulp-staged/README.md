# Gulp Staged

A gulp plugin that filters out unstaged files from streams

## Installation

```bash
npm install --save-dev gulp-staged@1.0.0-alpha.5
```
or
```bash
yarn add --dev gulp-staged@1.0.0-alpha.5
```

# Tslint Preset React

A pratical React style preset for TSLint

## Installation

```bash
npm install --save-dev @pshaw/tslint-preset-react tslint
```
or
```bash
yarn add --dev @pshaw/tslint-preset-react tslint
```

## Examples

```json
{
  "extends": [
    "@pshaw/tslint-preset-core",
    "@pshaw/tslint-preset-react"
  ]
}
```

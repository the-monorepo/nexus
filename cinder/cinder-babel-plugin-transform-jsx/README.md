# Babel Plugin Transform Jsx

Convert JSX logic to cinder compatible JavaScript

## Installation

```bash
npm install --save @cinder/babel-plugin-transform-jsx@1.0.0-alpha.0
```
or
```bash
yarn add @cinder/babel-plugin-transform-jsx@1.0.0-alpha.0
```

# Tester Mocha

Use Mocha tests in the fl test runner

## Installation

```bash
npm install --save @fault/tester-mocha@1.0.0-alpha.0 mocha
```
or
```bash
yarn add @fault/tester-mocha@1.0.0-alpha.0 mocha
```
